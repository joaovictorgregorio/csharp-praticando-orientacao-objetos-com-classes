## 📝 About This Repository

This repository contains the practical code developed during the course. Its main goal is to consolidate object-oriented concepts in C#, with emphasis on:

- Class creation
- Objects and instances
- Attributes and methods
- Encapsulation and access modifiers
- Constructors and behavior organization

## 💻 How to Run the Code

Since this project was built using the **.NET CLI**, you can easily run it from your terminal.

1. Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
2. Clone this repository and navigate to the desired challenge folder.
3. Run the following command in the terminal:

```bash
dotnet run
```
